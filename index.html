Installation Guide

Follow these simple steps to get DeepSeekManager up and running:

For Termux (Android)

git clone https://github.com/sudo-xda/DeepSeekManager.git

Clone the repository and run the installation script:

./DeepSeekManager.sh

For Raspberry Pi

git clone https://github.com/sudo-xda/DeepSeekManager.git

Clone the repository and run the installation script:

bash DeepSeekManager.sh
